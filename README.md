# Cubit

A modern speedcubing timer for iOS and Android. Built with Expo and React Native.

![Expo](https://img.shields.io/badge/Expo-50.0.0-blue)
![React Native](https://img.shields.io/badge/React%20Native-0.73-green)
![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue)

## Installation

```bash
# Clone the repository
git clone https://github.com/ThunderboltDev/cubit 
cd cubit

# Install dependencies
pnpm install

# Start the development server
pnpm expo start
```

## Running the App

- **iOS Simulator**: Press `i` in the terminal
- **Android Emulator**: Press `a` in the terminal  
- **Physical Device**: Scan the QR code with Expo Go
